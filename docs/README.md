### 1단계 요구사항

1. 지하철 역 관리 API 기능 완성
    - 지하철 역 생성 시 이미 등록된 이름으로 요청한다면 에러 응답
2. 지하철 노선 관리 API 구현
    - 지하철 노선 등록
        - 등록된 이름으로 요청하면 에러 응답
    - 지하철 노선 목록
    - 지하철 노선 조회
        - 조회된 노선이 없는 경우 에러 응답
    - 지하철 노선 수정
        - 중복된 이름으로 수정 시 에러 응답
    - 지하철 노선 삭제
3. End to End 테스트 작성
